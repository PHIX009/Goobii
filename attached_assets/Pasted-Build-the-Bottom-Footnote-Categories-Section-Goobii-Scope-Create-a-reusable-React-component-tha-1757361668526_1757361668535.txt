Build the Bottom “Footnote Categories” Section (Goobii)
Scope

Create a reusable React component that renders the three bottom categories: Services, Company, and Contact Us. Titles must be bold/thick, subtitles underneath, and Contact items must include icons. Content sits at the bottom of pages, evenly spaced, with left/right indents (side gutters).

Tech

Next.js (App Router) + React + TypeScript

Tailwind CSS

Inline SVG placeholders for icons (no external images)

Brand System (use exact values)

--brand-primary: #006980; /* main text, primary accent */

--brand-secondary: #003B5C; /* contrast accents */

--brand-bg: #FAFCFE; /* section background */

--brand-pop: #FF8200; /* focus/interactive pop */

--brand-contrast-2: #DCC57F;/* selection/highlight */

Fonts:

Ghost for large headings (if needed)

Grandview Bold for section titles (use as default here)

Grandview for all subtitle/body links

Provide @font-face placeholders and fallbacks (system-ui)

Component

Create components/FooterCategories.tsx with this structure and styling:

Wrapper <footer> with background var(--brand-bg), top border var(--brand-secondary) at 10–15% opacity.

Container: centered max width with side indents
class="max-w-screen-2xl mx-auto px-6 md:px-10 lg:px-16"
(keeps space on both sides at all breakpoints)

Vertical spacing: pt-12 pb-16

Grid: grid grid-cols-1 md:grid-cols-3 gap-8 md:gap-10

Column titles:

Tag: <h3>

Font: Grandview Bold, text-lg md:text-xl, font-extrabold, tracking-tight, text-[var(--brand-secondary)]

Subtitle links:

Tag: <a> within <ul><li> lists

Font: Grandview, text-[var(--brand-primary)], text-sm md:text-base

Hover/Focus: underline grows from center; text tint to var(--brand-secondary); visible focus ring ring-2 ring-[var(--brand-pop)] ring-offset-2

Contact icons:

Inline SVG placeholders left of each label (24px), aria-hidden="true", paired with accessible link text.

Icon tint: default var(--brand-secondary) at ~80% opacity; on hover/focus tint to var(--brand-primary)

Accessibility & polish

Title/links must be keyboard-navigable with visible focus rings (--brand-pop).

Icons are decorative; aria-hidden="true".

Ensure link hit targets ≥ 40px height on mobile (py-1.5 + line height achieves this).

All text colors must pass WCAG AA against --brand-bg.

Acceptance Criteria

Three columns render with bold/thick titles and subtitle links underneath.

Contact Us shows icons next to WhatsApp, Email, Phone.

Section is evenly spaced at the bottom and indented left/right (px-6 md:px-10 lg:px-16 with max-w-screen-2xl).

Colors/fonts match brand tokens; hover/focus states implemented.

Fully responsive: single column on mobile, three columns from md: and up.
