## Design System (Use EXACT Values)

### Colors (CSS Variables)

```
--brand-primary: #006980;  /* main color for all text and buttons */
--brand-secondary: #003B5C;/* secondary color to create contrast */
--brand-bg: #FAFCFE;       /* background + text inside buttons */
--brand-pop: #FF8200;      /* special contrast / time-related elements */
--brand-contrast-2: #DCC57F; /* secondary contrast for selections */
```

* Apply a light theme on `--brand-bg`.
* **Primary buttons:** background `--brand-primary`, text `--brand-bg`.
* **Secondary buttons or accents:** use `--brand-secondary` as needed.
* Use `--brand-pop` sparingly for time-sensitive badges, notices, or limited offers.
* Use `--brand-contrast-2` primarily for **selection states** (e.g., selected tab/card borders or highlights).

### Fonts (Exact)

* **Main titles:** **Ghost**
* **Subtitles:** **Grandview Bold**
* **Body & other text:** **Grandview**
* Implement via `@font-face` with local `fonts/` placeholders and robust fallbacks (e.g., Inter, system-ui) until the real font files are provided.
* Tailwind config: set font families to match the above usage.

### Components & Style

* Clean, informative layout; grid-based sections; rounded-2xl cards; soft shadows; generous padding.
* Simple inline SVG icon set for consistency.
* Motion: minimal hover/active states only.
